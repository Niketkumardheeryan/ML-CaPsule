## Sudoku-Solver
**version** - `1.1.2`

This software solve the sudoku puzzle using convolutions neural network.

### Requirements

- **`python`** - `3.7`
- **`keras`** -  `2.4.3`
- **`tensorflow`** -  `2.2.0`

---
### Project Content


| File      | Description |
| :-----------: | :-----------: |
| **sudoku.ipynb** | notebook for running and testing the project |
| **model.py**   | neural network implementation |
