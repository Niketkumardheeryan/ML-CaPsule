# Interactive Data Cleaning Challenge

## Overview
This project helps beginners practice data cleaning in Python. Users will fill missing values in the dataset using various methods: 'mean', 'median', or 'mode'.

## How to Use
1. Clone this repository.
2. Open the `interactive_data_cleaning_challenge.ipynb` notebook.
3. Follow the instructions in the notebook to clean the data by filling missing values.

## Requirements
Make sure you have the following Python packages installed:
- pandas
- numpy

You can install them using pip:

```bash
pip install pandas numpy
